What is Rocket Code ?

Rocket Code Camp is a one-month training program organized by Gistec. It focuses on teaching HTML, CSS, JavaScript, along with specialized training in Geographic Information Systems (GIS). This program is designed to equip participants with practical skills and knowledge in front-end web development and GIS technology.
